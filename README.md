# Personal Expense Tracker

A simple, lightweight web application built to track daily expenses. 

## Features
- **Add Expenses:** Track name, amount, and category.
- **Persistent Data:** Uses `LocalStorage` to save data across browser refreshes.
- **Dynamic Calculation:** Automatically updates the total spent.
- **Delete Functionality:** Remove individual entries easily.

## Tech Stack
- **Frontend:** HTML5, CSS3
- **Logic:** Vanilla JavaScript (ES6+)
- **Storage:** Browser LocalStorage API

## How to Run
1. Clone the repository.
2. Open `index.html` in any web browser.
